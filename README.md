# Pentafreight Frontend

A frontend recreation of the Pentafreight website with responsive design and smooth GSAP animations.

---

## Live Demo

https://penta-freight-41c7kt3t2-dilipp04s-projects.vercel.app

## Project Overview

This project recreates the following pages from the Pentafreight website:

- Home Page
- Industries Page
- Pentakuhl Page

The objective was to achieve:

- Pixel-perfect UI implementation
- Fully responsive layouts
- Smooth scrolling experience
- Modern animations using GSAP
- Reusable React components
- Clean and maintainable code structure

Challenges Faced

- Recreating complex scroll interactions
- Maintaining responsiveness across devices
- Matching animation timing and easing
- Building reusable animation components

---
